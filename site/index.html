Airways' Float Night Mods

Here are some simple mods for the casual / chill suvival game Float Night on Steam.

The mods:

CleanUp
Removes all the junk from the starting area.

Installation:

1. Install BepInEx in your Steam\steamapps\common\Float Night folder. It should have a new folder named BepInEx with a sub-folder named plugins amongst others.
2. Install ScriptLoader.dll to Steam\steamapps\common\Float Night\BepInEx\Plugins
3. Create a new Scripts folder next to Float Night's main program XuanYe.exe
4. Download these files to Scripts, and make sure they have a .cs extension and no extra extension (if you don't know how to do that, read turn off hide extensions in explorer first then make sure the filenames look exactly as they appear on this page)
