Airways' Float Night Mods

Here are some simple mods for the casual / chill suvival game Float Night on Steam.

The mods:

CleanUp.cs — Screenshot
Removes all the junk from the starting area.

MeatIsFood.cs
Makes meat dropped by enemies edible — but it only restores hunger, not health. Note that this will prevent a built in feature from working so don't install it until you're sure you want it.

ZAirwaysLoader.cs
Note: This MUST be installed for the other mods to work and must be the last mod loaded!

Installation:

1. Install BepInEx in your Steam\steamapps\common\Float Night folder.
    ※ This should have a new folder named BepInEx with a sub-folder named plugins amongst others.
2. Install ScriptLoader.dll to Steam\steamapps\common\Float Night\BepInEx\Plugins
3. Create a new Scripts folder next to Float Night's main program XuanYe.exe
4. Download these files to Scripts, and make sure they have a .cs extension and no extra extension.
    ※ If you don't know how to do that, read turn off hide extensions in explorer first then make sure the filenames look exactly as they appear on this page.

Other Notes:

※ Please visit my Float Night forum thread for any questions or issues!
